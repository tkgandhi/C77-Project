Visual Studio Code

Visual Studio Code is a platfrom on which you can perform text-based coding. You can start coding by 1st getting the app on your computer. Next, you can go to your folders and name it. When you will open Visual Studio, in the top-left corner will be File. Hover your mouse over it and click on Open Folder. After your folder will open, you can add an HTML, CSS, and/or JS file.

The console is used for finding errors code. You can open the console by right-clicking on the screen. Click on inspect and that will open something on the right. Then you can presss console. You can write code for console by creating a variable, and then writing the code 'console.log(x);'.
